# Example Projects

- `jshell-smoke-project`  
  JShell / run-main / console stress checks.

- `uml-relationships-project`  
  UML parsing/layout checks (interfaces, abstract classes, inheritance, enum, associations).

- `structogram-stress-project`  
  Structogram rendering checks (if/switch/loops/try-catch/finally/nesting).

- `reload-disk-project`  
  Reload-from-disk behavior checks for externally modified source files.

- `code-highlighting-project`  
  Scope/highlight rendering checks (nested blocks, javadocs, comments, selection behavior).
