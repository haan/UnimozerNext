# Structogram Stress Project

Use this project to validate structogram rendering for mixed and nested control flow.

## Files
- `src/StructogramCases.java` (methods with if/switch/loops/try-catch/finally)
- `src/StructogramMain.java` (simple runner)

## Quick check
1. Open `examples/structogram-stress-project` as folder project.
2. Compile project.
3. Open `StructogramCases` and move the caret through each method.
4. Confirm structogram updates and remains readable for nested blocks.
