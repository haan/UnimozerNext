# Array Display Project

Use this project to validate array display in the UML diagram and object bench.

## Quick Check
1. Open `examples/array-display-project` as a folder project.
2. Compile the project.
3. Confirm the UML diagram shows array field types such as `int[]`, `int[][]`, `String[]`, and `ArrayDisplayItem[]`.
4. Create an `ArrayDisplayCases` object with the no-argument constructor.
5. Confirm object-bench fields show compact previews for primitive, object, nested, empty, null-containing, inherited, and large arrays without widening the object card.
6. Run `ArrayDisplayMain` and confirm the console output includes:
   - `Array display cases ready`
   - `largeNumbers length: 32`
   - `matrix rows: 3`
